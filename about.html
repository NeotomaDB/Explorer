Neotoma Explorer is an interactive application that provides easy access to the Neotoma Paleoecology database through a web browser. Use it to: find data through searches on spatial, temporal, metadata criteria; browse and visualize live data and metadata; and, get data and information in a variety of formats (e.g. maps, graphs, exports). For more information, view the documentation.
