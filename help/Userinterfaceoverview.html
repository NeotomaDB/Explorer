User interface overview

Get started ››

 

  1. Application toolbar: open and/or launch main tools and application windows
  2. Messages: view status of application activities
  3. Map controls:  pan and zoom around the map
  4. Site inspector:  click on a mapped site to view summary metadata; access detailed datasets matching search criteria as well as other datasets associated with a site
  5. Search:  query the database for datasets that meet specified spatial, temporal, and metadata criteria
  6. Map:  view search results according to the locations of the sites with which they are associated
  7. Tables:  view search results in tabular format
  8. Dataset inspector:  view detailed data spreadsheet and associated metadata, chronologies, publications, and diagrams
  1. Dataset tray:  save datasets of interest for later viewing and download
  2. Search list:  manage display of search results as map layers and tables; combine search results with AND/OR operations into new layers
  1. Search properties:  change name, shape, size, and color of map symbols for a search
  2. Advanced taxa options:  search for taxa names by partial matching to a specified string
