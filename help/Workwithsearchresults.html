Work with search results

  • When a search generates results from the database, metadata about the matching datasets is returned to Neotoma Explorer where it can be viewed on the map as a layer and as a table.
  • You can keep as many search result sets (i.e., tables/layers) as you wish in an Explorer work session and manage them through the Search List tools.
  • Search result sets can be combined with AND/OR logic to create new sets.
  • The Dataset Inspector enables you to select a dataset and view its detailed data spreadsheet and associated metadata, chronologies, publications, and diagrams.
