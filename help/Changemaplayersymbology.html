Change map layer symbology

Work with search results ›› View results on the map ››

  1. Click the button to open the Current Searches list.
  2. Click a search layer color sample to open the Search Properties.
  3. Modify the desired properties and click the Apply button to make the change(s).
    1. edit label to identify search in layer list and table tab title
    2. select marker shape
    3. select marker size: small, medium, large
    4. select marker color
  4. When done, close the Search Properties window.
