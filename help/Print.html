Print

The print feature formats a print-ready page of either the current map view, or the selected result table.


Map

  1. Before printing, pan and zoom the map to your area of interest.  Optionally, use the Search Properties tools to change symbology of layers.  You can toggle the display of individual search layers in the Search List.
  2. Click the button in the main application toolbar.
  3. Click the text prompt to type a title if desired.
  4. Print the page using your Web browser's print function.


Table

  1. Select the search results table you wish to print.
  2. Click the button in the main application toolbar.
  3. Optionally, click the text prompt to type a title.
  4. Print the page using your Web browser's print function.
