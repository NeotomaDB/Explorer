System requirements

Get started ››

  • Neotoma Explorer has been tested in the following web browsers:
    • Google Chrome 33, 34
    • Mozilla Firefox 28, 29
    • Microsoft Internet Explorer 9,10
    • Apple Safari (Win) 5.1.7
  • you must enable JavaScript in your web browser
  • you may need to disable browser pop-up blocking for apps.neotomadb.org
  • we strongly recommend disabling the Skype Click-to-Call browser add-on/extension if installed; our testing has revealed that it severely impacts performance of large dataset spreadsheets/tables


For the best user experience, we recommend Google Chrome and Mozilla Firefox.
